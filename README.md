# Fundamentals of Data Analisys
This repository contains all the material related to the course "Fondamenti di Analisi dei dati" (Fundamental of Data Analisys) held in University of Catania. Since the course is in Italian, some of the material could be written in Italian.

## Root Directory
Contains the main project files, including Python scripts and notebooks. It also hosts the top-level folders for code, datasets, and lecture materials (note: lecture materials are written in Italian).

## codes/
Collection of Python scripts used for data analysis, plotting, and statistical examples. Each file represents a self-contained example or visualization.

## dataset/
Contains the dataset(s) used in the project. Typically includes CSV files for demonstrations and exercises.

## dispense/ *(Italian lecture notes)*
Folder containing LaTeX source files for the course handouts and chapters.  
Includes two important subdirectories:

### dispense/images/
All images (PNG, JPEG) used inside the LaTeX documents.
